# Express

## Download

- [Express](http://expressjs.com/fr/)
> Express is a minimal and flexible Node.js web application framework that provides a robust set of features for web and mobile applications.

```
npm i express
```

## Embed middleware

- compression
> Compress the page in gzip, send it more quickly to the navigator.

- cookie-parser
> Allows to manupilate cookies


- cookie-session
> Allows to manupilate sessions


- serve-static
> Allows to send static files (img, pdf, ...)


- serve-favicon
> Allows to send the favicon


- csrf
> Protect against the csrf hacks
