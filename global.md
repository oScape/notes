# Global information


## Versioning

- "0.3.5"
> Version 0.3.5 only

```
"dependencies": {
    "markdown": "0.3.5"
}
```

- "~0.3.5"
> OK for 0.3.5, 0.3.6, 0.3.7, ... versions until 0.4.0 not included

```
"dependencies": {
    "markdown": "~0.3.5"
}
```

- "~0.3"
> OK for 0.3.X, 0.4.X, 0.5.X versions until 1.0.0 not included

```
"dependencies": {
    "markdown": "~0.3"
}
```
